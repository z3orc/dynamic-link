Dynamic Link

An easy and lightning fast (âš¡) way of downloading a specific Minecraft-server JAR!

Instead of manually finding the download link to a particular JAR, ./mc-dynamic-link will automatically find the download link and redirect you. The JARs are found using the official APIs and downloaded directly from the official sources. The project supports downloading JARs from Mojang, PaperMC, and Purpur.

This project is currently in closed testing and therefore unavailable to the general public. The current plan is to release the project sometime in 2022, however, the date is still TBD.

Browser

Type the URL https://dynamic.z3orc.com/*flavour*/*version*/ into the addressbar of your browser, then fill in which flavour (that would be vanilla, paper or purpur) and the Minecraft version you prefer. You will then be redirected to the download url and your download should start shortly.

Command-line

Using WGET: WGET https://dynamic.z3orc.com/*flavour*/*version*/

Using CURL: CURL https://dynamic.z3orc.com/*flavour*/*version*/

Before running switch out flavour and version with your desired flavour (vanilla, paper or purpur) and Minecraft version.
